## Data BB Beteiligung

### institutions: names, id and bboxes

`institutions_bboxes.json` in this folder is committed in-repo and used at runtime by the BB survey. Update it manually when institution data changes (e.g. from FMC drive CSV: UTF-8 encoded, no non-German special characters).

Note: This data was needed for names, id and bboxes. The respective form fields for BB (read only field and SurveyMapLine) are not migrated to tanstack form.
